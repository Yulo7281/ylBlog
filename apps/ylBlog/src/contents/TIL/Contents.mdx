    ## :not() 伪类

    <TIL.ItemTags>
      <TIL.ItemTag>css</TIL.ItemTag>
      <TIL.ItemTag>选择器</TIL.ItemTag>
    </TIL.ItemTags>

    使用 `:not()` 伪类需要注意技巧，稍有不慎结果可能不符合预期。

    场景：为内联代码 `code` 设置样式，但需排除其父元素是 `pre` 或 `.code-block` 的情况。

    <TIL.DnD>
      <TIL.Dont>
        ```css {copy:false} {footer:false}
        :not(pre), :not(.code-block) {
          > code {
            background-color: 'red';
          }
        }
        ```
      </TIL.Dont>
      <TIL.Do>
        ```css {copy:false} {footer:false}
        :not(pre, .code-block) {
          > code {
            background-color: 'red';
          }
        }
        ```
      </TIL.Do>
    </TIL.DnD>

    <br/>
    错误解读：这种写法实际会匹配 父元素不是 pre 的所有元素 或 父元素不是 .code-block 的所有元素，导致选择范围过大（逻辑上是"或"关系而非"且"）。
